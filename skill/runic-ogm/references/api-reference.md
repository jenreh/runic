## Imports

Everything public is re-exported from the `runic.ogm` package root:

```python
from runic.ogm import (
    Node, Edge, Field, Relation,            # modeling
    Session, AsyncSession,                  # unit of work
    Repository, AsyncRepository,            # collection reads
    select, QueryBuilder,                   # query builder
    count, avg, sum_, min_, max_, collect,  # aggregation helpers
    Vector, GeoLocation,                    # native types
    DatetimeConverter, EnumConverter, TypeConverter,
    OrmError, EntityNotFoundError, DetachedEntityError,
    LazyLoadError, FieldValidationError, MetadataError,
    metadata, get_metadata,
)
from runic.ogm.driver.factory import create_driver
from runic.ogm.driver.falkordb import FalkorDBDriver, AsyncFalkorDBDriver
```

## Declaring models

```python
class User(Node, labels=["User"], primary_label="User"):
    ...

class Authored(Edge, type="AUTHORED"):
    ...
```

| Class keyword | On | Meaning | Default |
|---|---|---|---|
| `labels=[...]` | `Node` | Graph labels applied to the vertex | `[ClassName]` |
| `primary_label="..."` | `Node` | Label used for polymorphic MATCH/decoding | first label |
| `type="..."` | `Edge` | Relationship type string | ClassName |

- `__init__` is generated automatically and is **keyword-only**.
- Bare annotations (`name: str`) become `Field()`s; optional annotations
  (`x: T | None`) default to `None`, non-optional ones are required.
- Subclasses inherit parent fields; declare extra `labels` for polymorphism.
- All Node/Edge subclasses auto-register in the global `metadata` singleton.

## Field() parameters

`Field(*, ...) -> Any` — declares a scalar property. (Returns `Any` so
`name: str = Field()` type-checks cleanly.)

| Parameter | Type | Default | Purpose |
|---|---|---|---|
| `default` | `Any` | `MISSING` | Default value; mutually exclusive with `default_factory` |
| `default_factory` | `() -> Any` | `None` | Callable producing the default (e.g. `list`, `datetime.now`) |
| `init` | `bool` | `True` | Include in generated `__init__` |
| `kw_only` | `bool` | `True` | Keyword-only (always true in practice) |
| `index` | `bool` | `False` | Declare a range index on this property |
| `index_type` | `"FULLTEXT" \| "VECTOR" \| None` | `None` | Back fulltext / vector search |
| `unique` | `bool` | `False` | Declare a uniqueness constraint |
| `required` | `bool` | `False` | Mark as required (validation hint) |
| `primary_key` | `bool` | `False` | Mark as the primary key |
| `converter` | `TypeConverter \| None` | `None` | Explicit value converter (overrides auto) |
| `generated` | `bool` | `False` | Value generated by the DB; defaults to `None` until persisted |
| `interned` | `bool` | `False` | Store via FalkorDB `intern()` (string dedup) |

PK resolution: a field with `primary_key=True`, else a field named `id` by
convention.

## Relation() parameters

`Relation(*, ...) -> Any` — declares a relationship field on a `Node`.

| Parameter | Type | Default | Purpose |
|---|---|---|---|
| `relationship` | `str` | **required** | Relationship type in the graph (e.g. `"AUTHORED"`) |
| `direction` | `"OUTGOING" \| "INCOMING" \| "BOTH"` | **required** | Edge direction from this node's perspective |
| `target` | `str \| type` | **required** | Related Node class (string name or class) |
| `edge_model` | `str \| type \| None` | `None` | `Edge` model carrying edge properties |
| `cascade` | `bool` | `False` | Cascade saves to related entities |
| `lazy` | `bool` | `True` | Load on first access (sync only) vs require `fetch=` |
| `default` | `Any` | `None` | Default attribute value before loading |
| `default_factory` | `() -> Any` | `None` | Callable default |
| `init` | `bool` | `True` | Include in generated `__init__` |

- Single relation: annotate `Target | None`. Collection: annotate `list[Target]`.
- Relationship fields may **not** set `index`/`unique`.
- Relations are optional at construction (excluded from required kwargs).

## Native types & converters

| Type | Cypher wrapper (FalkorDB) | Read-back | Notes |
|---|---|---|---|
| `datetime` | — (ISO-8601 string) | `datetime` | `DatetimeConverter` auto-applied; keep tz-aware |
| `Enum` / `StrEnum` subclass | — (stored value) | enum member | `EnumConverter` auto-applied |
| `Vector` (a `list` subclass) | `vecf32($v)` | `Vector` | embeddings; **FalkorDB-only** |
| `GeoLocation(latitude, longitude)` | `point($v)` | `GeoLocation` | geo coords; **FalkorDB-only** |
| `str` + `Field(interned=True)` | `intern($v)` | `str` | dedups repeated strings; **FalkorDB-only** |

Auto-converters are assigned only when `converter=` is not set and the field is
not a relationship. Explicit `converter=` always wins. Write a custom converter
by subclassing `TypeConverter` (`to_graph`, `from_graph`, optional `cypher_fn`).

## Session

`Session(driver)` — synchronous unit of work. Use as a context manager.

| Method | Signature | Description |
|---|---|---|
| `add` | `(entity) -> None` | Stage an insert |
| `add_all` | `(entities: list) -> None` | Stage many inserts |
| `delete` | `(entity) -> None` | Stage a delete |
| `get` | `(cls, pk, fetch=None) -> entity \| None` | Read by PK; `fetch=[...]` eager-loads relations |
| `flush` | `() -> None` | Execute pending/dirty/deleted now (no commit boundary) |
| `commit` | `() -> None` | `flush()` then clear tracking |
| `rollback` | `() -> None` | Discard pending state / abort transaction |
| `expire` | `(entity) -> None` | Mark for reload on next access |
| `refresh` | `(entity) -> None` | Re-read entity from the graph now |
| `expunge` | `(entity) -> None` | Detach a single entity |
| `expunge_all` | `() -> None` | Detach all |
| `relate` | `(source, field, target, edge=None) -> None` | MERGE a relationship; `field` may be a string or descriptor; `edge` is an `Edge` instance |
| `unrelate` | `(source, field, target) -> None` | Delete a relationship |
| `execute` | `(cypher, params=None, write=False) -> GraphResult` | Raw Cypher |
| `scalars` | `(stmt) -> list[T]` | Run `select()` → entities |
| `scalar` | `(stmt) -> T \| None` | Run `select()` → one or none |
| `count` | `(stmt) -> int` | Run `select()` → row count |
| `all_rows` | `(stmt) -> list[dict]` | Run a `project()`/`aggregate()` statement |
| `all_with_edges` | `(stmt) -> list[tuple]` | Run a traversal with `return_edge` |
| `query` | `(cls) -> QueryBuilder` | Session-bound builder (terminal methods on it) |
| `fulltext_search` | `(cls, *, query, fields=None) -> FulltextQueryBuilder` | FalkorDB fulltext |
| `vector_search` | `(cls, *, field, vector, k=10) -> VectorQueryBuilder` | FalkorDB KNN |
| `close` | `() -> None` | Release the session |

Dirty tracking: mutating any field on a loaded (persistent) entity sets
`_dirty`, so the next `commit()` emits a `SET`. Construction does **not** mark
dirty. Identity map: `get()` returns the same object per `(type, pk)` within a
session.

## AsyncSession

`AsyncSession(async_driver)` — full async parity. `get`, `flush`, `commit`,
`rollback`, `refresh`, `relate`, `unrelate`, `execute`, `scalars`, `scalar`,
`count`, `all_rows`, `all_with_edges`, `close` are **coroutines** (`await` them).
`add`/`add_all`/`delete`/`expire`/`expunge` are sync. `query`,
`fulltext_search`, `vector_search` return async builders (use `await qb.all()`).

**No lazy loading:** accessing an unloaded relation raises `LazyLoadError`. Use
`await session.get(cls, pk, fetch=[...])`. See
[../examples/async_session.py](../examples/async_session.py).

## Repository

`Repository(session, cls)` (and `AsyncRepository`). Collection-oriented reads
plus a raw-Cypher escape hatch.

| Method | Signature | Description |
|---|---|---|
| `find_all` | `(skip=0, limit=None, order_by=None, desc=False) -> list[T]` | All entities, optional paging/order |
| `find_all_by_ids` | `(ids: list) -> list[T]` | Batch fetch by PK |
| `count` | `() -> int` | Total entities of this type |
| `exists` | `(pk) -> bool` | PK existence check |
| `query` | `() -> QueryBuilder` | Session-bound builder for this type |
| `cypher` | `(q, params=None, returns=None, write=False) -> list` | Raw Cypher; `returns=Cls`/`int` maps rows |
| `cypher_one` | `(q, params=None, returns=None) -> T \| None` | First mapped row |
| `cypher_raw` | `(q, params=None) -> GraphResult` | Unmapped result (`.columns`, `.rows`) |

Subclass `Repository[T]` to add domain methods; build on `self.query()` or
`self.cypher(...)`.

## QueryBuilder

Created by `select(Cls)` (session-free) or `session.query(Cls)` (session-bound).

**Construction / refinement** (chainable, return the builder):

| Method | Description |
|---|---|
| `where(expr, on=None)` | Add a filter; `on="alias"` targets an aliased node/edge. Repeated calls AND-combine |
| `alias(name)` | Name the current pattern element for traversal/filtering |
| `traverse(rel, edge_alias=None, optional=True)` | One hop along a `Relation`; `optional=False` = required join |
| `repeat(rel, min_hops=1, max_hops=None)` | Variable-length path |
| `with_(*aliases)` | `WITH` pipeline boundary |
| `order_by(field, desc=False)` | Sort |
| `limit(n)` / `skip(n)` | Page |
| `distinct()` | `DISTINCT` in the RETURN |
| `project(*fields)` | Return scalar columns (read via `all_rows`); keys like `"n.name"` |
| `aggregate(*exprs, group_by=None)` | Aggregations. `group_by` is kept verbatim in RETURN: pass `"alias.property"` (e.g. `"n.city"`) to group by a field, or a bare alias to group by the whole node |
| `return_target(alias)` | Decode this alias as the result entity |
| `return_nodes(*aliases)` / `return_edge(alias)` | Columns for `all_with_edges` |

**Terminal methods** (on a session-bound builder from `session.query()`):
`all()`, `one()`, `count()`, `scalar()`, `scalars()`, `all_rows()`,
`all_with_edges()`. For `select()` statements call the equivalent
`session.<method>(stmt)` instead.

**Inspect:** `build() -> (cypher: str, params: dict)` — generate Cypher without
executing.

## Expressions & aggregation helpers

Field-descriptor operators (class-level access) produce filter expressions:

`==`, `!=`, `<`, `<=`, `>`, `>=`, `.contains(s)`, `.startswith(s)`,
`.endswith(s)`, `.matches(regex)`, `.is_null()`, `.is_not_null()`,
`.in_(list)`, `.not_in_(list)`. Combine with `&` (AND), `|` (OR), `~` (NOT) —
**parenthesize each operand**: `(A.x == 1) & (A.y == 2)`.

Aggregation helpers (each returns an expr with `.as_("alias")`):

| Helper | Cypher | Notes |
|---|---|---|
| `count("*")` / `count(field, distinct=False)` | `count(...)` | `"*"` for rows |
| `avg(field)` | `avg(...)` | |
| `sum_(field)` | `sum(...)` | trailing underscore avoids shadowing builtins |
| `min_(field)` / `max_(field)` | `min/max(...)` | |
| `collect(field, distinct=False)` | `collect(...)` | gather into a list |

Read aggregate/projection results with `all_rows()` (list of dicts keyed by the
alias or `"n.<field>"`).

## Drivers & factory

```python
from runic.ogm.driver.factory import create_driver

create_driver("falkordb", host="localhost", port=6379, graph="app")
create_driver("neo4j",    host="localhost", port=7687, database="neo4j",
              username="neo4j", password="…")          # encrypted=True default
create_driver("memgraph", host="localhost", port=7687, database="memgraph",
              username="…", password="…")              # encrypted=False default
create_driver("arcadedb", host="localhost", port=7687, database="db",
              username="root", password="…")           # plaintext bolt
create_driver("age",      host="localhost", port=5432, database="postgres",
              graph="app", username="postgres", password="…")  # psycopg3
```

- All drivers expose `.close()`. Pass the driver to `Session(driver)`.
- Embedded FalkorDB for tests: `FalkorDBDriver(FalkorDB(protocol=2).select_graph("app"))`.
- Async FalkorDB: `AsyncFalkorDBDriver(async_graph_handle)`.
- `intern`/`vecf32`/`point` (Vector/GeoLocation/interned) are **FalkorDB-only** native wrappers; other backends store the raw value. Fulltext and vector search work on FalkorDB, Neo4j¹, and Memgraph¹ but not on ArcadeDB or AGE.
  ¹ Requires pre-created named indexes and (Memgraph) MAGE modules.
- Full backend capability matrix in [cookbook.md](cookbook.md#multi-backend).

## Exceptions

All inherit from `OrmError`.

| Exception | Raised when |
|---|---|
| `EntityNotFoundError` | A required entity is missing |
| `DetachedEntityError` | Lazy-accessing a relation on a detached/expunged entity |
| `LazyLoadError` | Lazy relation access inside an `AsyncSession` (use `fetch=`) |
| `FieldValidationError` | A field value fails validation |
| `MetadataError` | Model/metadata misconfiguration (unknown target, etc.) |
